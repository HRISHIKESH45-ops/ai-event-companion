# 🎥 2-Minute Demo Video Script

**Target Length:** ~120 seconds.
**Vibe:** Energetic, professional, and fast-paced. 

---

### [0:00 - 0:15] Introduction (Screen: Hero Section & API Key)
* **Visual:** Show the top of the website. Mouse hovering over the shiny title. Type in your API key to show the "Key Active" green badge flip on.
* **Audio script:** "Welcome to the AI Event Companion. Hackathons and conferences are chaotic, so we built an intelligent, multi-modal web app that acts as your personal AI concierge. It uses Google's Gemini Flash models, but forces strict JSON schema outputs and features a heavily localized Bring-Your-Own-Key system."

### [0:15 - 0:40] The Setup (Screen: Event Form & Image Upload)
* **Visual:** Scroll down to the form. Select 'The Hustler' persona. Drag and drop the `sample_hackathon_flyer.png` into the flyer box.
* **Audio script:** "Instead of typing out schedules manually, watch this. I'm just going to upload this image of a hackathon flyer. Gemini's Vision capabilities will literally "read" the agenda for us. I'll select 'The Hustler' persona because I want to focus on networking, hit 'Generate', and let the AI do the heavy lifting."

### [0:40 - 1:10] The Output & Features (Screen: Generated Timeline)
* **Visual:** Scroll down to the generated timeline. Hover over the Hype Badge. Point the mouse to the colored Energy Heatmap bars.
* **Audio script:** "Boom. Instantly, we have a perfectly structured schedule. Notice three things: First, the **Hype Badge** up top dynamically adopts the tone of our Hustler persona. Second, the AI generated a customized **Preparation Guide** based on the events. Third, the AI specifically graded every activity to create this visual **Energy Heatmap**, so we know exactly when our brain might burn out."

### [1:10 - 1:40] AI Copilot & Tweak (Screen: Chatbot at the bottom)
* **Visual:** Scroll to the bottom "Ask AI to Tweak" box. Type: *"Add a 45-minute power nap at 2 PM"*, then hit Send. Wait for the UI to update.
* **Audio script:** "But schedules change. So we built a dynamic **Tweak Copilot**. I can just tell the AI: 'Add a 45-minute power nap at 2 PM'. The backend intelligently modifies the existing JSON structure without breaking the app context, and hot-swaps the timeline in place."

### [1:40 - 2:00] Outro & Code Resilience (Screen: Code / GitHub)
* **Visual:** Briefly show the `Download Schedule` calendar button, then switch to your VS Code showing `fallback` logic in `route.ts`.
* **Audio script:** "Finally, you can 1-click export this directly to Google Calendar. Under the hood, this is Hackathon-proof: we implemented a multi-model failover system. If Gemini 2.5 hits a quota limit, it instantly falls back to 2.0 without the user ever noticing. Thank you for checking out AI Event Companion!"
